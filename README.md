## v-image-preview

vue图片预览插件

### 效果

![demo](./img/demo.gif)

### 使用方法：
1. 安装包
```
npm install --save v-image-preview 
```

2. 引入并使用：
```js
import VImagePreview from 'v-image-preview';
vue.use(VImagePreview);
```

3. 在img标签上使用v-preview指令
```html
<img  v-preview src="https://mock.jpg"/>
```
